## 部署说明

### 快速部署

使用Docker Compose一键部署：
```bash
docker-compose up -d
```

### 手动部署

1. **服务器端**
```bash
cd server
./gradlew bootRun
```

2. **Web界面**
```bash
cd web
python -m http.server 8000
```

3. **Android客户端**
```bash
cd android
./gradlew assembleRelease
```

详见 `docs/DEPLOYMENT.md`

## API端点

### 设备管理
- `GET /api/devices` - 获取所有设备
- `GET /api/devices/{deviceId}` - 获取设备详情
- `POST /api/devices` - 注册设备
- `PUT /api/devices/{deviceId}/status` - 更新状态

### 位置管理
- `POST /api/positions` - 上报位置
- `GET /api/positions?deviceId={id}` - 查询位置历史

### 脚本管理
- `GET /api/scripts` - 获取所有脚本
- `POST /api/scripts` - 创建脚本
- `DELETE /api/scripts/{id}` - 删除脚本
- `GET /api/scripts/pending?deviceId={id}` - 获取待执行脚本
- `POST /api/scripts/results` - 上报执行结果
- `GET /api/scripts/results` - 查询执行结果

详见 `docs/API.md`
