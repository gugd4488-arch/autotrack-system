# AutoTrack System

整合 Auto.js 自动化 + Traccar GPS 追踪的统一系统

## 项目组成

1. **Auto.js 自动化** - Android JavaScript 自动化工具
2. **Traccar GPS** - 实时位置追踪系统  
3. **统一后台** - 集成管理服务器

## 快速部署

```bash
# 服务器
cd server && ./gradlew bootRun

# Android
cd android && ./gradlew assembleRelease
```

## 功能

- ✅ 远程脚本执行
- ✅ GPS 实时追踪
- ✅ 统一设备管理
- ✅ Web 控制面板
